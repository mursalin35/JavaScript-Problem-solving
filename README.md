# JavaScript Problem Solving

This repository contains solutions for 10 basic JavaScript problems.  
All solutions are written in **plain JavaScript** using loops, conditionals, arrays, and string methods.  

## Problems Solved

1. Reverse a String
2. Count Vowels in a String
3. Check for Palindrome
4. Find the Maximum Number
5. Remove Duplicates from an Array
6. Sum of All Numbers in an Array
7. Find Even Numbers in an Array
8. Capitalize First Letter of Each Word
9. Find the Factorial of a Number
10. PingPong Challenge

## How to Run

1. Make sure [Node.js](https://nodejs.org/) is installed.
2. Open terminal and navigate to the project folder.
3. Run the file using:

```bash
node index.js
